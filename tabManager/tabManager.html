This section provides powerful tools for organizing your tabs and windows.

Move tab right: L

Windows Overview

  • Use ↑ (Up) / K: Select a window above.

  • Use ↓ (Down) / J: Select a window below.

  • Press Enter: Move the current tab to the selected window.

The window list order is: New Empty Window, Other Windows, then the Current Window where the popup is open.
