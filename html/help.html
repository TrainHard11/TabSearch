Your Smart Tab & Bookmark Guide

Welcome! This extension helps you find and manage your tabs and favorite websites super fast.

1. Quick Search (Main View)

This is where you usually start. Just type to instantly find any open tab or your saved "Marks."

  • Smart Results: Your personal "Marks" appear first if they match an open tab, keeping your favorite sites easy to spot.
  • Instant Access: Hit Enter to switch to a selected tab or open a Mark. No results? Press Enter again for a quick Google search (you can turn this off in Settings).
  • Persistent Search: Your last search stays active. Just start typing or hit Backspace to begin a new one.

2. Extension Views & Navigation

Jump between different tools using the Function (F) keys:

  • F1: Settings

    Customize how the extension works:

    • Enable/disable Google search if no results.
    • Choose if Marks show in main search.
    • Set up custom URLs for quick tab moving (Ctrl + 1-4).

    Press F1 again to go back to Search.

  • F2: My Bookmarks (Marks)

    Your personal, quick-access bookmarks:

    • Save the current tab as a new Mark.
    • Edit or remove your saved Marks.

    Press F2 again to go back to Search.

  • F3: Harpoon

    Your essential tabs, always at your fingertips:

    • Add the current tab to your Harpoon list.
    • Reorder, activate, or remove Harpooned tabs.

    Press F3 again to go back to Search.

  • F4: Help

    You're here! This guide helps you get the most out of the extension.

    Press F4 again to go back to Search.

  • F5: Keyboard Shortcuts (Chrome)

    Opens Chrome's shortcut page where you can customize global keys for this extension.

    This will close the popup.

3. Keyboard Shortcuts

Speed up your workflow with these handy shortcuts:

A. Global Shortcuts:

  • Ctrl + 1 to Ctrl + 4: Move selected tab/Mark to a custom URL (set in Settings).

  • F1: Open/close **Settings**.

  • F2: Open/close **My Bookmarks (Marks)**.

  • F3: Open/close **Harpoon**.

  • F4: Open/close **Help**.

  • F5: Open Chrome's extension shortcuts page.

B. In Quick Search View:

  • ↑ (Up) / Alt + K: Move selection up.

  • ↓ (Down) / Alt + J: Move selection down.

  • Enter: Activate selected item or Google search.

  • Delete / Ctrl + D: Close selected **tab**.

  • Ctrl + Shift + D: Close **all** filtered tabs.

  • Typing / Backspace / Delete: Clear previous search and start new.

C. In Harpoon View (F3):

  • ↑ (Up) / K / Alt + K: Move selection up.

  • ↓ (Down) / J / Alt + J: Move selection down.

  • Enter: Activate selected Harpoon tab.

  • N / J (Shift+J) / Alt + N: Move selected Harpoon item **down**.

  • P / K (Shift+K) / Alt + P: Move selected Harpoon item **up**.

  • Ctrl + D: Remove selected item from Harpoon (doesn't close the tab).

Press F4 again to close help and return to search.
