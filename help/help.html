Fuzzy Tab & Search Navigator

This extension makes you faster in your browser! It's designed to help you quickly find tabs, manage your go-to sites, and navigate with ease.

1. Quick Search (Main View)

This is your starting point. Just type to instantly find any open tab or your saved "Marks."

  • Smart Results: Your personal "Marks" (bookmarks) appear first if they match an open tab, prioritizing your most important sites.
  • Instant Access: Hit Enter to switch to a selected tab or open a Mark. If your search yields no results, pressing Enter again can perform a quick Google search (you can adjust this in Settings).
  • Persistent Search: Your last search stays active. Simply start typing, or hit Backspace to begin a new search.

2. Extension Views & Navigation

Jump between different tools and sections using the Function (F) keys:

  • F1: Settings

    Customize how the extension behaves:

    • Web Search Navigator: This activates an arrow cursor for Google search results which allows quicker navigation through them.
    • Show Marks in Search: Choose whether your Marks should be searchable in the main Quick Search View.
    • Custom URL Links: Set up to 7 custom URLs that you can open instantly with configurable shortcuts (like Ctrl + 1). These menu is dedicated for link you visit all the time.
    • Exact Match for Marks: When checked, only Marks that are an "exact match" for the current tab's URL (ignoring things like "www." or "https://") will show up as a Smart Result. Unchecked, it will match more broadly.

  • F2: Bookmarks (Marks)

    Your personal, quick-access bookmarks. Use this view to save the current tab as a new Mark, or edit and remove your existing ones.

    • Add Bookmarks: Simply fill in the name and URL fields and click "Add Bookmark." URLs and names must be unique.
    • Bookmark Search: Press / (forward slash) to bring up a search bar. Type to instantly filter your bookmarks by name or URL.

      • ↓ (Down) / J / Alt + J: Navigate search results downwards.

      • ↑ (Up) / K / Alt + K: Navigate search results upwards.

      • Enter: Open the selected bookmark from search results.

      • Escape / Click outside: Close the search bar and clear results.

    • Include in Tab Search: Each bookmark now has a checkbox next to it. If checked, that specific bookmark will appear as a searchable item in the main Quick Search View. If unchecked (default for new bookmarks), it will only be visible within the Bookmarks (Marks) view.
    • Move Bookmarks: Use the ▲ (up) and ▼ (down) buttons next to each bookmark, or select an item and press Shift + K to move it up or Shift + J to move it down. Note: You cannot reorder bookmarks while the search bar is active.
    • Remove Bookmarks: Click the âœ• button next to a bookmark, or press d or Ctrl + D when a bookmark is selected, to remove it.

  • F3: Harpoon

    Your essential tabs, always at your fingertips for lightning-fast access. Harpoon holds up to 4 tabs, each tied to a specific global shortcut:

    • Alt + 1 to Alt + 4: Instantly jump to the tab assigned to that Harpoon slot.
    • Dynamic Links: If you reorder tabs within the Harpoon list using J/K, the shortcut (Alt + 1, etc.) will automatically follow the new tab in that slot. This means the keymap always accesses the tab *currently at that position* in your Harpoon list, enabling incredible speed.

  • F4: Help

    You're here! This guide helps you get the most out of the extension.

  • F5: Keyboard Shortcuts (Chrome)

    Opens Chrome's internal shortcut page where you can customize global keybindings for this and other extensions. This is useful for assigning your own preferred shortcuts to activate different features of the extension.

  • F6: Tab Management

    This view offers advanced control over your tabs and Chrome windows.

    • Move Tab Within Current Window:

      • Press 1-9: Move the current active tab to that specific 1st through 9th position.

      • Press H: Move the current tab one position to the left.

      • Press L: Move the current tab one position to the right.

    • Move Tab Between Windows (Teleport):

      This allows you to move the active tab to any other open Chrome window, or a brand new one.

      • Navigate windows: Use ↑ (Up) / K or ↓ (Down) / J to select a window from the list.

      • Execute move: Press Enter to move the current tab to the selected window. The extension popup will quickly close and re-open on the new target window/tab.

    • Window List Order: The list of windows is ordered as: New Empty Window (always first), followed by Other Windows, and finally the Current Window where the popup is open (at the bottom).

3. Keyboard Shortcuts

A. Global Shortcuts (Customizable in Chrome's Extension Shortcuts - F5):

These shortcuts work even when the extension popup isn't open. You can set your preferred keys for them via the F5 link. Here's what each command does:

  • Open_Main_View: Opens the extension's main popup, ready for you to start searching tabs.

  • Harpoon_open: Opens the Harpoon view directly, letting you quickly manage your essential tabs.

  • Harpoon_add: Adds your current tab to your Harpoon list.

  • Bookmark_add: Adds your current tab as a new bookmark (Mark).

  • Bookmarks_open: Opens the Bookmarks (Marks) view directly, letting you manage your saved links.

  • Harpoon_1, Harpoon_2, Harpoon_3, Harpoon_4: Instantly switch to the tab currently assigned to Harpoon slots 1 through 4, respectively.

  • Custom_Tab_1 through Custom_Tab_7: Opens the custom URL you've set up in the Settings for that specific slot.

  • Move_tab_to_1st, Move_tab_to_2nd, Move_tab_to_3rd, Move_tab_to_4th: Moves your current tab to the 1st, 2nd, 3rd, or 4th position in the current browser window's tab bar.

  • New_empty_tab: Opens a brand new, empty tab in your current window.

  • Cycle_tabs_playing_audio: Quickly cycles through all open tabs that are currently playing audio.

  • Cycle_YouTube_tabs: Cycles specifically through all open YouTube tabs.

  • Move_tab_left: Moves your current tab one position to the left in the tab bar.

  • Move_tab_right: Moves your current tab one position to the right in the tab bar.

  • Cycle_media_tabs: Cycles through all tabs that are playing any kind of media (like videos or music).

  • Open_Tab_Management_View: Opens the new Tab Management view directly.

B. In Quick Search View:

  • ↑ (Up) / Alt + K: Move selection up.

  • ↓ (Down) / Alt + J: Move selection down.

  • Enter: Activate selected item or Google search.

  • Delete / Ctrl + D: Close selected **tab**.

  • Ctrl + Shift + D: Close **all** filtered tabs.

  • Typing / Backspace / Delete: Clear previous search and start new.

C. In Harpoon View (F3):

  • ↑ (Up) / K / Alt + K: Move selection up.

  • ↓ (Down) / J / Alt + J: Move selection down.

  • Enter: Activate selected Harpoon tab.

  • Shift + J: Move selected Harpoon item **down** in the list.

  • Shift + K: Move selected Harpoon item **up** in the list.

  • Ctrl + d / d / Delete: Remove selected item from Harpoon (this *doesn't* close the tab itself).

Press F4 again to close help and return to the Quick Search view.
